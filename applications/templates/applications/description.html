backtesting-walkfoward

Backtesting é uma etapa fundamental para criação de estratégias de trading lucrativas a longo prazo. Para simplificar a criação de backtesting a backtesting-walfoward é uma biblioteca python que auxilia na criação de backtesting sobre data de ativos financeiros. Ela também faz walkfoward testing na qual se trata de vários backtestings em etapas. Esse projeto mostra bem a aplicação de OOP do python para resolver um problema real e complexo. Para rodar e entender como usar a biblioteca basta abrir o repositório no github e seguir o read-me. Qualquer dúvida ou sujestão de melhoria por favor entre em contato.

Python Birds

Python Birds um jogo simplificado do Angry Birds feita em python utilizando o tkinter como placa gráfica. O objetivo dele é ensinar os programadores conceitos de orientação a objetos e como utilizar o github na prática. Para isso o desenvolvedor deve fazer um fork do repositório original e aos poucos ir implementando as funcionalidades seguindo os testes existentes e ir realizando os commits.
