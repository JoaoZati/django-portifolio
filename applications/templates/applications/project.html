backtesting-walkfoward


Categoria: Projetos e Bibliotecas em Python


Backtesting é uma etapa fundamental para criação de estratégias de trading lucrativas a longo prazo. Para simplificar a criação de backtesting a backtesting-walfoward é uma biblioteca python que auxilia na criação de backtestings sobre data de ativos financeiros. Ela também faz walkfoward testing na qual se trata de vários backtestings em etapas aumento a probabilidade da estratégia ser mais coerente com a realidade. Esse projeto mostra bem a aplicação de OOP do python para resolver um problema real e complexo. Para rodar e entender como usar a biblioteca basta abrir o repositório no github e seguir o read-me. Qualquer dúvida ou sujestão de melhoria por favor entre em contato.
